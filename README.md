# Marketing Campaign EDA Analysis

This project performs Exploratory Data Analysis (EDA) on a marketing campaign dataset to identify business insights.

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn

## Analysis Performed
- Data cleaning
- Marketing channel distribution
- ROI comparison
- Marketing funnel analysis

## Insights
- Email marketing provides the highest ROI.
- Social media generates high awareness.
- Optimization in the conversion stage can improve performance.

## Files
- eda_analysis.py → Python analysis code
- marketing_campaign.csv → Dataset
- report.pdf → Business insights report
